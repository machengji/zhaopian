# TonePort - Lightroom 风格反色/色调迁移网页工具

一个纯前端（无后端依赖）的小工具，支持：

1. 上传参考图（分析色调）
2. 上传目标图（应用色调）
3. 自动迁移颜色特征
4. 前后对比预览 + PNG 下载

## 快速使用

1. 直接双击打开 `index.html`
2. 上传参考图和目标图
3. 选择算法并调整参数
4. 点击“开始迁移”
5. 拖动预览滑杆对比结果，点击“下载结果”

## 算法说明（参考开源项目思路）

- `Reinhard (LAB 均值/方差)`  
  对应经典 Reinhard 色彩迁移思路（LAB 空间统计匹配），参考：
  - chia56028/Color-Transfer-between-Images
  - ptran1203/color_transfer（同思路但更强调性能实现）

- `Enhanced (a/b 相关性增强)`  
  在 `a/b` 通道加入 2x2 协方差白化-着色变换，处理色彩相关性，思路借鉴：
  - TJCoding Enhanced 系列中强调的通道相关矩阵处理方向

## 参数

- `迁移强度`：0-100%，控制原图与迁移结果混合比例
- `饱和度`：0.2x-1.8x，控制迁移后 `a/b` 通道色彩幅度
- `先反色再做色调迁移`：模拟“反色 + 色调迁移”流程

## 技术实现

- `Canvas + TypedArray` 像素级处理
- `sRGB <-> XYZ <-> LAB` 手写转换
- `Reinhard` 统计匹配
- `2x2` 对称协方差矩阵特征分解（用于增强模式）
- 前后对比滑杆预览（Before/After）
