# TonePort - 轻量级 Web 色调迁移工具

TonePort 是一个面向浏览器的轻量化调色工具，定位是：

1. 一键上传即用，不依赖大型软件工作流
2. 移动端也可流畅预览
3. 免费可用 + 高级功能订阅扩展

项目地址：`https://machengji.github.io/zhaopian/`

## 当前能力

1. 上传参考图和目标图
2. 自动色调迁移（Reinhard / Enhanced）
3. 前后对比滑块预览
4. PNG 下载
5. 预设库（胶片柔和、电影青橙、电商清透、社媒高对比）
6. AI 推荐（启发式参数推荐）
7. 移动端性能档位（极速/均衡/高质量）
8. 后台高清导出（预览先出，高清后补）

## 这次重点优化

### 1) 手机浏览器卡顿

- 计算搬到 `Web Worker`，主线程不再被像素循环阻塞
- 采用“快速预览 + 后台高清导出”双通道流程
- 根据档位自动控制参与计算的像素量，避免 12MP 原图直接硬算

### 2) 迁移效果不稳定

- 使用稳健统计（带裁剪的 LAB 统计）降低极端像素干扰
- 新增亮度保真参数，保留目标图细节层次
- 新增风格预设后处理，使结果更接近可用成片

## 运行方式

1. 打开 `index.html`
2. 上传参考图和目标图
3. 选择算法、预设、性能档位
4. 点击“开始迁移”
5. 拖动滑块预览并下载结果

## 技术结构

- 前端：`index.html` + `style.css` + `app.js`
- 算法执行：`worker.js`（LAB 色彩迁移 + 增强相关性）
- 核心流程：
  1. 预览分辨率下快速计算
  2. UI 先可交互
  3. 高清结果后台生成并替换下载文件

## 目标用户场景

1. 电商产品图统一色调（高频、批量、要求稳定）
2. 社交媒体博主快速出片（速度优先）
3. 小型工作室低预算调色（可替代部分重型软件环节）
